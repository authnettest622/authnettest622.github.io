Partial Authorization
Help

Select this option if you are ready to submit partial authorizations. A partial authorization transaction is one in which two or more transactions are used to cover the total amount of the order. For example, a customer might present a gift card whose value does not cover the entire cost of a purchase.

MasterCard and Discover require support for partial authorizations, authorization reversals, and balance response for prepaid cards (such as debit or gift cards). This includes MasterCard, Discover, JCB, and Diners card programs in the U.S.

Before you can activate partial authorizations, you need to make sure your web site is set up to handle this type of transaction. For more information about setting up your web site to handle partial authorizations, see the documentation at the Developer Center

If you turn off this option and you submit an authorization request for a prepaid card whose available balance is less than the total purchase amount, the transaction will be declined due to lack of funds.
